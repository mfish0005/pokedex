Textarea Component

Multi-line text input component with multiple variants, sizes, and states for the Fish UI library.

Variants

Different visual styles based on your fish-styles theme.

variant="default"
variant="success"
variant="warning"
variant="danger"

Sizes

Different sizes for various use cases.

size="small"
size="medium"
size="large"

States

Different states for textarea inputs.

Normal state
[disabled]="true"
[readonly]="true"

Resize Options

Different resize behaviors for textarea.

resize="none"
resize="vertical"
resize="horizontal"
resize="both"

Character Limits

Textarea with character count and limits.

[maxLength]="100"
[minLength]="10"

Usage

Import

import { TextareaComponent } from 'fish-ui';

Basic Usage

<fish-textarea 
  placeholder="Enter your message"
  [rows]="4"
  [value]="textareaValue"
  (valueChange)="onTextareaChange($event)">
</fish-textarea>

With Variant and Size

<fish-textarea 
  placeholder="Success textarea"
  variant="success"
  size="large"
  [rows]="5"
  resize="vertical">
</fish-textarea>

Reactive Forms

<form [formGroup]="myForm">
  <fish-textarea 
    formControlName="description"
    placeholder="Enter description"
    [rows]="4"
    [maxLength]="500">
  </fish-textarea>
</form>
