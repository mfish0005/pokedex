Getting Started

Quick start guide to install and set up Fish UI in your Angular project.

Setup

After installation, you need to configure Fish UI in your Angular application.

1. Import Styles

Add the Fish UI styles to your global styles file:

src/styles.scss 
@import '~fish-ui/styles/fish-styles/all.scss';

2. Import Components

Fish UI provides standalone components that can be imported directly where needed. This approach works for both module-based and standalone Angular applications:

Standalone Component Example 
import {{ '{' }} Component {{ '}' }} from '@angular/core';
  import {{ '{' }} CommonModule {{ '}' }} from '@angular/common';
  import {{ '{' }} RouterOutlet {{ '}' }} from '@angular/router';
  
  // Import Fish UI components individually
  import {{ '{' }} ButtonComponent, CardComponent {{ '}' }} from 'fish-ui';
  
  @Component({{ '{' }}
    selector: 'app-root',
    standalone: true,
    imports: [
      CommonModule,
      RouterOutlet,
      ButtonComponent,  // Add components to imports
      CardComponent
    ],
    templateUrl: './app.component.html',
    styleUrls: ['./app.component.scss']
  {{ '}' }})
  export class AppComponent {{ '{' }}
    title = 'my-app';
  {{ '}' }}
Module-based Example 
import {{ '{' }} NgModule {{ '}' }} from '@angular/core';
  import {{ '{' }} BrowserModule {{ '}' }} from '@angular/platform-browser';
  import {{ '{' }} AppComponent {{ '}' }} from './app.component';
  
  // Import Fish UI components individually
  import {{ '{' }} ButtonComponent, CardComponent {{ '}' }} from 'fish-ui';
  
  @NgModule({{ '{' }}
    declarations: [
      AppComponent
    ],
    imports: [
      BrowserModule,
      ButtonComponent,  // Add components to imports
      CardComponent
    ],
    providers: [],
    bootstrap: [AppComponent]
  {{ '}' }})
  export class AppModule {{ '{' }} {{ '}' }}

Examples

Here are some common usage patterns:

Basic Button

src/app/app.component.html 
<fish-button variant="primary" (click)="handleClick()">
    Click me!
  </fish-button>

Card with Content

src/app/app.component.html 
<fish-card>
    <h3>Card Title</h3>
    <p>Card content goes here.</p>
    <fish-button>Action</fish-button>
  </fish-card>

Search Component

src/app/app.component.html 
<fish-search
    placeholder="Search..."
    (search)="onSearch($event)"
    (clear)="onClear()">
  </fish-search>
