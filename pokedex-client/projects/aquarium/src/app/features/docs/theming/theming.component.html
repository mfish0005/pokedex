Theming

Customize Fish UI with SCSS variables. Override any variable with !default before importing fish-styles.

Quick Theme Setup

Create a theme file and override any variables you need:

src/styles/themes/my-theme.scss 
// Most Common Color Overrides
$primary: #007bff;        // Main brand color
$secondary: #6c757d;      // Secondary color
$success: #28a745;        // Success states
$danger: #dc3545;         // Error states
$warning: #ffc107;        // Warning states

// Typography Overrides
$family-sans-serif: 'Inter', 'Segoe UI', sans-serif;
$family-primary: $family-sans-serif;
$size-6: 1rem;           // Base text size
$size-7: 0.875rem;       // Small text size

// Gray Scale Overrides
$gray-lightest: #f8f9fa;  // Very subtle backgrounds
$gray-lighter: #e9ecef;   // Card backgrounds
$gray-light: #dee2e6;     // Borders and dividers
$gray: #6c757d;           // Secondary text
$gray-dark: #495057;      // Icon colors
$gray-darker: #343a40;    // Placeholder text
$gray-darkest: #212529;   // Primary text

// Import fish-styles AFTER your overrides
@import '~fish-ui/styles/fish-styles/all';
example.component.scss 
// Import your theme where it's needed.
@import './themes/my-theme';

Usage Examples

Once themed, all components and helper classes automatically use your custom values:

Helper Classes (Automatic)

HTML - Text Colors 
<!-- Primary color variations -->
<p class="has-text-primary">Primary text</p>
<p class="has-text-primary-light">Primary light text</p>
<p class="has-text-primary-dark">Primary dark text</p>

<!-- Semantic colors -->
<p class="has-text-success">Success message</p>
<p class="has-text-danger">Error message</p>
<p class="has-text-warning">Warning message</p>
<p class="has-text-info">Info message</p>
HTML - Background Colors 
<!-- Background color variations -->
<div class="has-background-primary">Primary background</div>
<div class="has-background-success">Success background</div>
<div class="has-background-danger">Danger background</div>
<div class="has-background-warning">Warning background</div>
<div class="has-background-info">Info background</div>
HTML - Typography 
<!-- Font sizes -->
<h1 class="is-size-1">Hero heading (3rem)</h1>
<h2 class="is-size-2">Page heading (2.25rem)</h2>
<h3 class="is-size-3">Section heading (1.875rem)</h3>
<p class="is-size-6">Body text (1rem)</p>
<small class="is-size-7">Small text (0.875rem)</small>

<!-- Font weights -->
<p class="has-text-weight-light">Light weight (300)</p>
<p class="has-text-weight-normal">Normal weight (400)</p>
<p class="has-text-weight-semibold">Semibold weight (500)</p>
<p class="has-text-weight-bold">Bold weight (700)</p>

Components (Automatic)

HTML - Components inherit theme 
<!-- All components use your theme colors -->
<fish-button variant="primary">Primary Button</fish-button>
<fish-button variant="danger">Danger Button</fish-button>

Complete Theme Example

Here's a comprehensive theme file showing all overrideable variables:

src/styles/themes/complete-theme.scss 
// COLORS - Override base colors
$white: #ffffff;
$black: #1a1a1a;
$gray-lightest: #f8f9fa;
$gray-lighter: #f5f5f5;
$gray-light: #eeeeee;
$gray: #e0e0e0;
$gray-dark: #bdbdbd;
$gray-darker: #9e9e9e;
$gray-darkest: #757575;

// Base color palette
$green: #28a745;
$green-light: #51cf66;
$green-dark: #1e7e34;

$blue: #007bff;
$blue-light: #4dabf7;
$blue-dark: #0056b3;

$red: #dc3545;
$red-light: #ff6b6b;
$red-dark: #bd2130;

$orange: #fd7e14;
$orange-light: #ffd43b;
$orange-dark: #e8590c;

$yellow: #ffc107;
$yellow-light: #fff3cd;
$yellow-dark: #e0a800;

$purple: #6f42c1;
$purple-light: #a98eda;
$purple-dark: #4c0d95;

// SEMANTIC COLORS - For consistent theming
$primary: $blue;
$secondary: $gray;
$success: $green;
$warning: $orange;
$danger: $red;
$info: $blue;
$text: $gray-darkest;

// TYPOGRAPHY - Font families and sizes
$family-monospace: 'JetBrains Mono', 'Fira Code', monospace;
$family-sans-serif: 'Inter', 'Segoe UI', sans-serif;
$family-primary: $family-sans-serif;
$family-secondary: $family-monospace;

// Font sizes (responsive scale)
$size-1: 3rem;     // 48px - Hero titles
$size-2: 2.25rem;  // 36px - Page titles
$size-3: 1.875rem; // 30px - Section headings
$size-4: 1.5rem;   // 24px - Subsection headings
$size-5: 1.25rem;  // 20px - Large text
$size-6: 1rem;     // 16px - Base text
$size-7: 0.875rem; // 14px - Small text

// Font weights
$weight-light: 300;
$weight-normal: 400;
$weight-semibold: 500;
$weight-bold: 700;

// Size array for helper classes
$sizes: $size-1, $size-2, $size-3, $size-4, $size-5, $size-6, $size-7;

// COLOR MAPS - Used by helper classes (.has-text-*, .has-background-*)
$colorMap: (
  "white": $white,
  "black": $black,
  "primary": $primary,
  "secondary": $secondary,
  "success": $success,
  "warning": $warning,
  "danger": $danger,
  "info": $info,
  "green": $green,
  "blue": $blue,
  "red": $red,
  "orange": $orange,
  "yellow": $yellow,
  "purple": $purple,
  "gray": $gray,
);

// SHADE MAPS - Used for hover states and variations
$shadeMap: (
  "gray-lightest": $gray-lightest,
  "gray-lighter": $gray-lighter,
  "gray-light": $gray-light,
  "gray-dark": $gray-dark,
  "gray-darker": $gray-darker,
  "gray-darkest": $gray-darkest,
  "green-light": $green-light,
  "green-dark": $green-dark,
  "blue-light": $blue-light,
  "blue-dark": $blue-dark,
  "red-light": $red-light,
  "red-dark": $red-dark,
  "orange-light": $orange-light,
  "orange-dark": $orange-dark,
  "yellow-light": $yellow-light,
  "yellow-dark": $yellow-dark,
  "purple-light": $purple-light,
  "purple-dark": $purple-dark,
);

// Import fish-styles AFTER all overrides
@import '~fish-ui/styles/fish-styles/all';
