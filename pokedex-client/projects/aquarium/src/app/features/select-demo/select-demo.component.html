Select Component

A customizable dropdown select component with multiple variants, sizes, and states.

Basic Usage

Simple select with basic options and placeholder text.

Basic Select

Selected: {{ basicValue() || 'None' }}
<fish-select
  [options]="basicOptions"
  placeholder="Choose an option"
  (selectionChange)="onBasicChange($event)">
</fish-select>

Disabled Options

Individual options can be disabled while keeping the select enabled.

Mixed Options

const options = [
  {{ '{' }} value: 'option1', label: 'Available Option' {{ '}' }},
  {{ '{' }} value: 'option2', label: 'Disabled Option', disabled: true {{ '}' }},
  {{ '{' }} value: 'option3', label: 'Another Available Option' {{ '}' }}
];

Usage

The select component provides a flexible dropdown interface with keyboard navigation and accessibility features.

options
SelectOption[]
Array of options to display in the dropdown
placeholder
string
Placeholder text when no option is selected
variant
'default' | 'outline' | 'filled'
Visual style variant
size
'small' | 'medium' | 'large'
Size of the select component
disabled
boolean
Whether the select is disabled
required
boolean
Whether the select is required
error
boolean
Whether to show error state
errorMessage
string
Error message to display
