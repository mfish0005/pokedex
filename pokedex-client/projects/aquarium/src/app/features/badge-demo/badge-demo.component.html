Badge Component

Small status indicators and labels with multiple variants and sizes.

Variants

Colors generated from your fish-styles theme

Primary variant="primary"
Secondary variant="secondary"
Success variant="success"
Warning variant="warning"
Danger variant="danger"
Info variant="info"

Sizes

Different badge sizes for various contexts.

Small size="small"
Medium size="medium"
Large size="large"

Styles

Different visual styles and shapes.

Default Default style
Rounded [rounded]="true"
Outline [outline]="true"
Both [rounded]="true" [outline]="true"

Custom Colors

Use custom colors for colors outside your theme

Custom Orange variant="custom" customColor="#ff6b35"
Custom Purple variant="custom" customColor="#6c5ce7"
Custom Teal variant="custom" customColor="#00b894"
Custom Rounded Custom with rounded

Usage

Import

import { BadgeComponent } from '@fish-ui/components';

Template

<fish-badge 
  variant="primary" 
  size="medium"
  [rounded]="true"
  [outline]="false">
  Badge Text
</fish-badge>

<!-- Custom color badge -->
<fish-badge 
  variant="custom"
  customColor="#6c5ce7"
  size="small">
  Custom
</fish-badge>
