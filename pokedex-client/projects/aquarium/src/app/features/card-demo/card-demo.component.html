Card Component

Flexible container component with elevation, padding, and interaction options.

Elevation Levels

Different shadow depths to create visual hierarchy.

Small Elevation

Subtle shadow for gentle separation

 elevation="small"

Medium Elevation

Moderate shadow for clear distinction

 elevation="medium"

Large Elevation

Strong shadow for prominent display

 elevation="large"

Padding Options

Control internal spacing for different content types.

Small Padding

Compact spacing for dense layouts

 padding="small"

Medium Padding

Balanced spacing for most content

 padding="medium"

Large Padding

Generous spacing for important content

 padding="large"

Interactive States

Cards can respond to user interactions.

Static Card

No interaction states

 Default behavior

Hoverable Card

Elevation increases on hover

 [hoverable]="true"

Clickable Card

Shows pointer cursor and click effects

 [clickable]="true"

Interactive Card

Both hover and click effects

 [hoverable]="true" [clickable]="true"

Usage

Import

import { CardComponent } from 'fish-ui';

Basic Template

<fish-card 
  elevation="small" 
  padding="medium">
  <h3>Card Title</h3>
  <p>Card content goes here.</p>
</fish-card>

Interactive Card

<fish-card 
  elevation="small" 
  padding="medium"
  [hoverable]="true"
  [clickable]="true"
  (click)="handleCardClick()">
  <h3>Interactive Card</h3>
  <p>Click me for action!</p>
</fish-card>
