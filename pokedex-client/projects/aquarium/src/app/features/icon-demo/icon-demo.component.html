Usage

Import

import { IconComponent } from 'fish-ui';

Basic Usage

<fish-icon name="fish"></fish-icon>

With Size and Color

<fish-icon 
  name="heart" 
  size="large" 
  color="#ff4444">
</fish-icon>

Custom Size

<fish-icon 
  name="star" 
  [size]="48">
</fish-icon>
