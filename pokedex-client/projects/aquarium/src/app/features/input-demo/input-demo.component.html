Input Component

Versatile input component with multiple variants, sizes, and states for the Fish UI library.

Variants

Different visual styles based on your fish-styles theme.

variant="default"
variant="success"
variant="warning"
variant="danger"

Sizes

Different sizes for various use cases.

size="small"
size="medium"
size="large"

States

Different states for form inputs.

Normal state
[disabled]="true"
[readonly]="true"

Input Types

Different HTML input types supported.

type="text"
type="email"
type="password"
type="number"
type="url"
type="search"

Usage

Import

import { InputComponent } from 'fish-ui';

Basic Usage

<fish-input 
  placeholder="Enter your name"
  [value]="inputValue"
  (valueChange)="onInputChange($event)">
</fish-input>

With Variant and Size

<fish-input 
  placeholder="Success input"
  variant="success"
  size="large"
  type="email">
</fish-input>

Reactive Forms

<form [formGroup]="myForm">
  <fish-input 
    formControlName="email"
    type="email"
    placeholder="Enter your email">
  </fish-input>
</form>
